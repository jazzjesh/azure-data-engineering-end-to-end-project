# Raw Data Reference (Bronze Layer)

The full AdventureWorks dataset used in this project is sourced from:

https://github.com/anshlambaoldgit/Adventure-Works-Data-Engineering-Project/tree/main/Data

Due to size constraints, the complete raw data files are not stored in this repository.

## Contents Included in the Source:
- AdventureWorks_Calendar.csv
- AdventureWorks_Customers.csv
- AdventureWorks_Product_Categories.csv
- AdventureWorks_Product_Subcategories.csv
- AdventureWorks_Products.csv
- AdventureWorks_Returns.csv
- AdventureWorks_Sales_2015.csv
- AdventureWorks_Sales_2016.csv
- AdventureWorks_Sales_2017.csv
- AdventureWorks_Territories.csv

These files represent the **Bronze (raw ingestion)** layer in ADLS Gen2.
This project uses smaller sample files stored in `datasamples/` for demonstration.
